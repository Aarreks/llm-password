# Experiment 1D: token-DP semantic continuation scoring

This folder contains the Experiment 1D notebook and the 100-password input CSV used for the checked-in Experiment 1D result files.

The notebook evaluates exact token-DP continuation probabilities under the selected language-model prompts. The result folders under `results/exp1d_32-32-32/` and `results/exp1d_512-512-512/` contain the corresponding saved outputs.
