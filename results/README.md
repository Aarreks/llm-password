# Results included

This folder contains selected reports and experiment outputs used for the final project presentation.

## Demo reports

- `random_autoaccept_report.txt`: random-looking password autoaccept example.
- `math_structured_accept_report.txt`: math/programming-looking password accepted but not autoaccepted.
- `code_fragment_reject_interactive_run.txt`: common code fragment rejected by LLM-DP semantic predictability.
- `stage0_stable_substring_rejects_report.txt`: stable substring Stage 0 rejecting a weak compositional family.

## Experiment 2

`exp2_template_sensitivity/` contains continuation-template sensitivity outputs.

## Experiment 2B negative result

`exp2b_direct_rating_negative_result/` contains the direct JSON-rating experiment. This is included as a negative result: directly asking the LLM whether a password is structured is not the final method.
