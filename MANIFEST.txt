LLM Password Router repo manifest

This repository contains the final demo code, supporting experiments, selected results, report/design artifacts, and AI interaction evidence for the ECE 202C LLM password-security project.

Extracting from zip:
This zip is flat at the repository root. Extract its contents directly into the cloned Git repository folder that contains `.git`.

Top-level files:

* .gitignore
* MANIFEST.txt
* README.md
* requirements.txt
* report.pdf
* design.pdf
* AI_INTERACTION.md

Main demo notebooks:

* notebooks/final_password_router_demo.ipynb
  Original presentable router demo using uncapped LLM token-DP.
* notebooks/capped_final_password_router_demo.ipynb
  Experimental capped router variant using capped DP with brute-force-style fallback edges.

Source code:

* src/password_router_demo.py
  Python code extracted from the final demo notebook.

Data:

* data/README.md
* data/stage0_bad_substrings.csv
  Stage 0 Aho-Corasick resource used by the demo notebook.

Scripts:

* scripts/generate_stage0_bad_substrings.py
  Reproducibility/provenance script for generating a similar Stage 0 pattern list. The checked-in CSV is the one used by the demo.

Notes:

* The final router notebooks expect one raw password per line.
* Do not add split markers.
* The original demo and capped demo are kept side by side because the capped version is an experimental policy variant.
* Experiment 1D is a controlled continuation experiment; the final router scores whole raw passwords from prefix length 0.
* Direct JSON rating is included as a negative result and is not used by the final router.
